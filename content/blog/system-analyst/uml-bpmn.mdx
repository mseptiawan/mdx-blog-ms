# UML dan BPMN dalam Analisis Sistem

Dalam analisis sistem, UML dan BPMN sering digunakan untuk memodelkan proses
dan struktur sistem.

## UML (Unified Modeling Language)
UML digunakan untuk:
- Use Case Diagram
- Class Diagram
- Sequence Diagram

## BPMN (Business Process Model and Notation)
BPMN fokus pada:
- Alur proses bisnis
- Aktivitas dan decision
- Stakeholder non-teknis
