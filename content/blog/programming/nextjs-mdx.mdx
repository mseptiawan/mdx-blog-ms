# Membuat Blog MDX dengan Next.js

MDX memungkinkan kita menulis Markdown dan React dalam satu file.

## Kenapa MDX?
- Fleksibel
- Cocok untuk blog teknis
- Mudah dikembangkan

## Struktur Project
- app/ → routing
- content/ → MDX
- public/ → asset
