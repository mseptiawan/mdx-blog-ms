# Dasar System Requirement Specification (SRS)

System Requirement Specification (SRS) adalah dokumen formal yang
mendefinisikan kebutuhan sistem secara detail dan terstruktur.

## Apa itu SRS?
SRS berfungsi sebagai:
- Acuan pengembangan sistem
- Dokumen komunikasi antara stakeholder
- Dasar pengujian sistem

## Struktur Umum SRS
1. Pendahuluan
2. Deskripsi umum
3. Kebutuhan fungsional
4. Kebutuhan non-fungsional
